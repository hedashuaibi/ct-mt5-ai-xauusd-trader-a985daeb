# Testing & Research Verification

## Research Reporting UX Verification
Institutional research reports are verified using Playwright to ensure accessibility and visual consistency across generated HTML artifacts.

Key UX standards:
- **Contrast:** Secondary colors must meet WCAG AA standards (>= 4.5:1).
- **Navigation:** `:focus-visible` must be implemented for all interactive elements.
- **Interactions:** Scroll-reveal behaviors and animations must be tested for performance and usability.
