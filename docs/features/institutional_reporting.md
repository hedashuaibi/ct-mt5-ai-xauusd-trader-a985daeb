# Institutional Research Reporting

The institutional reporting system generates high-quality strategy summaries for portfolio managers, quant leads, and auditors.

## Key Features
- **KPI Dashboard**: High-level metrics for resilience, stability, efficiency, and diversification.
- **Decision Status**: Clear 'VERIFIED', 'PROVISIONAL', or 'REJECTED' status for strategy deployment.
- **Actionable Recommendations**: Explicit list of required changes or optimizations.
- **Multi-domain Coverage**: Integrated reports covering regimes, stress tests, hyperopt, and model drift.

## Report Sections
1. **Executive Summary**: Strategic overview of strategy health.
2. **KPI Ribbon**: Instant visual feedback on critical performance scores.
3. **Market Regime Analysis**: Performance breakdown by market condition.
4. **Stress Test Outcomes**: Resilience scores under adversarial scenarios.
5. **Hyperparameter Robustness**: Stability of optimal parameters.
6. **Trade Pattern Findings**: Behavioral insights and signal motifs.
7. **Model Drift Observations**: Feature and distribution shift detection.
8. **Conclusion & Recommendations**: Clear deployment path and required actions.

## Usage
Reports can be generated via the `ResearchOrchestrator` and exported to HTML or Markdown using the `ResearchReporter`.
